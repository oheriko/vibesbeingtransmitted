# Roadmap

Ideas for future development. Not prioritized yet.

## Core (MVP)
- [ ] Spotify status in Slack - show currently playing track/artist
- [ ] Open in Spotify - teammates can click to open the music

## Social Discovery
- [ ] **Music Match** - "You and @sarah both listened to Radiohead 12 times this week"
- [ ] **Team Vibes Digest** - Weekly summary of team listening activity
- [ ] **Quick Reactions** - Emoji react to someone's current track
- [ ] **"Listening With"** - See who else is playing the same track right now

## Collaborative Features
- [ ] **Channel Playlists** - Each Slack channel gets a collaborative Spotify playlist
- [ ] **Office Radio** - Shared queue that anyone can add to
- [ ] **Listening Party** - Sync playback across teammates for remote coworking

## Privacy & Control
- [ ] **Blocklist** - Hide specific tracks/artists from sharing
- [ ] **Schedule Sharing** - Only share during certain hours
- [ ] **Focus Mode** - One-click pause sharing
- [ ] **Genre Filter** - Only share certain genres

## Platform Expansion
- [ ] Apple Music support
- [ ] YouTube Music support
- [ ] Tidal support
- [ ] Slack Huddle integration - show music when in a huddle

## Fun Stats
- [ ] **Taste Compatibility** - Compatibility scores between teammates
- [ ] **Genre Breakdown** - Personal and team listening analytics
- [ ] **Streak Tracking** - Sharing streaks and milestones
